## Using
Use **only Resource Pack & Data Pack** to add some new Jukebox Songs in *Minecraft: Java Edition*, no Mod.

### Steps
1. Start your game.  
2. Open *Resource Pack UI* and drag gthe Resource Pack into the window.  
3. Create new world , open *Data Pack UI* and drag the Data Pack into the window; or open your *World folder*, open `./datapack` folder, copy the Data Pack into it.
4. Use added recipes to craft Music Discs.  
5. Place the Music Discs into a jukebox to listen to music
