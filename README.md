## Using Easily
Use **only Resource Pack & Data Pack** to add some new Jukebox Songs in *Minecraft: Java Edition*, no Mod.

First, start your game. Second, open *Resource Pack UI* and drag the Resource Pack into the window. Third, create new world , open *Data Pack UI* and drag the Data Pack into the window; or open your *World folder*, open *./datapack* folder, copy the Data Pack into it
